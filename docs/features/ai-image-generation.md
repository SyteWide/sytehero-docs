# AI Image Generation (fal.ai)

Generate hero images directly in WordPress using AI-powered image-to-image transformation. Select a source image from your media library, describe the look you want, and create stunning hero visuals without leaving the admin.

## Setup

1. Go to **SyteHero → Integrations**
2. Find the **fal.ai** card
3. Get an API key from [fal.ai](https://fal.ai) and paste it into the API Key field
4. Toggle **Enabled** and click **Save Settings**
5. Click **Test Connection** to verify your key works

## Using AI Generation

The **AI** button appears wherever you can select a hero image:

- **Product edit screen** — in the SyteHero metabox, next to "Select Hero Media"
- **Heroes tab** — next to each slide's media selector
- **Schedules tab** — when adding slides that support hero images

### Workflow

1. Click the **AI** button to open the generation modal
2. **Select a source image** — click the image area or "Change Image" to pick from your media library
3. **Write a prompt** — describe the transformation you want (e.g., "Transform this product into a floating scene with cosmic nebula background, dramatic lighting")
4. **Choose a model** — the dropdown shows all available image-to-image models with cost indicators
5. **Adjust strength** — lower values keep closer to the original; higher values allow more creative freedom
6. Click **Generate** and wait for the progress bar to complete
7. Review the result — click **Regenerate** to try again with the same settings, or adjust and re-generate
8. Click **Use This Image** to save the generated image to your media library and assign it to the slide

### Cost Indicators

Each model shows a cost tier next to its name:

| Tier | Approximate Cost |
|------|-----------------|
| $    | ~$0.01–0.02/image |
| $$   | ~$0.02–0.05/image |
| $$$  | ~$0.05–0.10/image |
| $$$$ | ~$0.10+/image |

The exact per-image price appears below the model dropdown when selected. Costs are charged by fal.ai to your fal.ai account — SyteHero does not add any markup.

### Tips

- **Start with a clear product photo** — the AI works best with well-lit, clean source images
- **Be specific in your prompts** — "product floating in cosmic nebula with purple and blue tones, cinematic lighting" works better than "make it look cool"
- **Use lower strength** (30–50%) for subtle enhancements, **higher strength** (70–90%) for dramatic transformations
- **Try different models** — each model has its own style and strengths

## AI Studio: Media Manager

The **AI Studio** tab (SyteHero admin → AI Studio) provides tools for managing all AI-generated images.

### AI Media

Browse, inspect, and clean up every image generated by SyteHero's AI tools.

- Each card shows: thumbnail, model name, cost tier badge, two-line prompt preview, generation date, and an **In Use** / **Unused** status badge.
- **Select** images with the checkbox overlay to bulk-select.
- **Delete Selected** removes unused items (images currently referenced anywhere on your site are protected).
- **Backfill** — If you generated AI images before updating to this version, click **Tag Pre-existing AI Images** to retroactively tag them so they appear in this view.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| "AI" button doesn't appear | Verify the integration is enabled in Settings → Integrations |
| "Failed to load models" | Check your API key is valid; click Test Connection |
| Generation times out | Try a different model or simplify your prompt |
| Image quality is poor | Use a higher-quality source image; try a more expensive model tier |
